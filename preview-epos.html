📦 Project Overview

Web-based cashier application for retail stores or MSMEs to record sales transactions, manage stock and view daily reports.

🛠️ Key Features

  • Multi-user login (admin and cashier)
  • Recording sales transactions
  • Print receipt
  • PDF reports
  • Responsive UI

💻 Technology

  • HTML, CSS, JavaScript
  • Bootstrap
  • Laravel
  • MySQL Database
